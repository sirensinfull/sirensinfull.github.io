## DECISIONS MADE (book-relevant)

1. **File format discovery.** The .docx and .odt files in the project are NOT binary Office documents. They are plain UTF-8 text files with misleading extensions. Readable with `cat`. This matters for any future tooling.

2. **Addendum A merged into XIX.** Paper XIX and its separate Addendum A (lattice shards, phantom bit-flip) combined into a single document. The addendum material became §8–§11 of the expanded paper.

3. **Papers XX–XLIII not bulk-expanded.** The originals (3–4k words each for XX–XXVII, 1.5–4k for XXVIII–XLIII) were already well-written. The rebuild added standardized headers, dependency chains, and classification tags without padding. This was a deliberate call — the content was already there.

4. **Paper XXVI ("A Letter to Humanity") preserved as personal document.** Not reformatted into technical style. Header standardized, voice preserved.

5. **Classification taxonomy settled.** FOUNDATIONAL / APPLIED / THEORETICAL / CORRELATIVE / SPECULATIVE / THESIS / COMPLETION / COMEDIC TRUTH / PERSONAL. Each paper tagged. This is the sorting axis for the books.

6. **Dependency graph threaded.** Every paper carries "Dependencies" (what it needs) and "Required by" (what needs it). The graph is forward-and-backward navigable.

7. **Five-phase structure confirmed.** Foundational (I–X) → Applied Physics (XI–XVIII) → Encoding & Information (XIX–XXVII) → Cryptography & Cognition (XXVIII–XXXIV) → Completion (XXXV–XLIII). This is the table of contents axis.
